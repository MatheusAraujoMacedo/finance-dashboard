# Finance Dashboard — Projeto para Portfólio

Estrutura mínima (Vite + React) para rodar localmente.

## Como usar

1. Instale dependências:
   ```bash
   npm install
   ```

2. Rode em desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra o browser no endereço informado pelo Vite (ex: http://localhost:5173).

> Observações:
> - Este projeto usa o CDN do Tailwind para facilitar pré-visualização. Para produção, recomendo configurar o Tailwind localmente.
> - Dependências principais: `react`, `react-dom`, `recharts`, `vite`.
